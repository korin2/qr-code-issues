# QR Generator — бесплатный генератор QR-кодов

! Репозиторий предназначен исключительно для фиксации ошибок от пользователей !
---

📱 **https://qr.2korin.ru/** — сервис для создания красивых QR-кодов прямо в браузере. 
Все данные обрабатываются локально, ничего не отправляется на сервер.

## Возможности

### 📋 Типы QR-кодов
| Тип | Описание |
|-----|----------|
| 🔗 URL / Ссылка | Любая ссылка на сайт |
| 📝 Текст | Любой произвольный текст |
| 📞 Телефон | Номер телефона (tel:) |
| ✉️ Email | Email с опциональной темой письма |
| 💬 SMS | SMS-сообщение |
| 🟢 WhatsApp | Чат в WhatsApp |
| 🔵 Telegram | Чат в Telegram |
| 📶 Wi-Fi | Автоподключение к Wi-Fi сети |
| 👤 Визитка (VCard) | Контактные данные для сохранения |

### 🎨 Настройки
- Цвет QR-кода
- Цвет фона
- Размер (от 200 до 800px)
- 🖼️ **Загрузка логотипа** в центр QR-кода

### ⬇️ Экспорт
- PNG — растровое изображение
- SVG — векторное изображение
- 📋 Копирование в буфер обмена

### 🎯 Дополнительно
- 🌙 **Тёмная тема** — автоматически подстраивается под систему
- 📜 **История** — последние 10 QR-кодов (сохраняется в браузере)
- 🔄 Быстрое восстановление из истории
